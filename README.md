
## Instrucciones de Uso

### Modo Producto Único:
1. Selecciona "Single Product"
2. Completa: Descripción, MCC, Item#, Sub-total
3. Haz clic en "Generate Ticket"
4. Confirma para imprimir

### Modo Múltiples Productos:
1. Selecciona "Multiple Products"
2. Usa "+ Add Product" para agregar más items
3. Completa los datos para cada producto
4. Haz clic en "Generate Multi-Product Ticket"
5. Confirma para imprimir

### Funciones Adicionales:
- **Print Preview**: Vista previa antes de imprimir
- **Reset**: Limpiar todos los campos
- **Live Preview**: Vista en tiempo real del ticket

## Requisitos Técnicos
- Navegador moderno con JavaScript habilitado
- Impresora térmica de 80mm (recomendado)
- Conexión a Internet para Print.js

## Personalización
- Modifica `TAX_RATE` en `script.js` para cambiar el impuesto
- Ajusta los estilos en `style.css`
- Cambia el logo reemplazando `Logo.jpeg`

## Solución de Problemas
1. **No imprime:**
   - Verifica que la impresora esté conectada
   - Permite ventanas emergentes
   - Usa Print Preview primero

2. **Cálculos incorrectos:**
   - Verifica que los números sean válidos
   - Recarga la página

3. **Diseño roto:**
   - Verifica la conexión a Print.js
   - Limpia la caché del navegador
